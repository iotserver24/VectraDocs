# Welcome to Your Docs

This is a template for creating a documentation site using [Fumadocs](https://fumadocs.vercel.app).

## How to use

1.  **Edit Content**: Modify `content/docs/index.mdx` to change this page.
2.  **Add Pages**: Create new `.mdx` files in `content/docs`.
3.  **Customize Layout**: Edit `app/layout.tsx` or `app/(home)/page.tsx` to change the look and feel.

## Next Steps

Explore the `content/docs` directory and start adding your own content!
